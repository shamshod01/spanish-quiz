# 🇪🇸 스페인어 단어 퀴즈 (KIIP 2~6과)

스페인어 수업 2~6과의 단어와 표현 **564개**(전공 68개 포함)를 뽑아 만든 웹 퀴즈 게임입니다.

## 기능
- 과별 / 분야별 필터 (동사, 빈도부사, 요일, 전공, 색깔, 숫자, tener 표현 등)
- 스페인어→한국어, 한국어→스페인어, 섞기
- 4지선다 · 직접 입력(체크 버튼) · 플래시카드 모드
- 🔊 **발음 듣기** (스페인어 음성, 자동 재생 옵션)
- 연속 정답 스트릭, 점수, 틀린 단어만 다시 풀기

## 파일
- `index.html` — 퀴즈 앱 (단독 실행 가능)
- `qr.png` / `qr.svg` — 접속용 QR 코드
